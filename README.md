# Plataforma Inmobiliaria Dinámica

Aplicación web para la gestión de activos inmobiliarios, desarrollada durante la formación especializada en iPartek[cite: 1, 2].

## 🚀 Características principales
* **Interfaz Dinámica:** Desarrollo de una SPA (Single Page Application) centrada en la experiencia de usuario[cite: 1, 2].
* **Consumo de APIs:** Integración de servicios backend para la gestión de datos en tiempo real[cite: 1, 2].
* **Arquitectura Limpia:** Uso de patrones de diseño para un código escalable y mantenible[cite: 1, 2].

## 🛠️ Stack Tecnológico
* **Frontend:** Angular, JavaScript, HTML5 / CSS3[cite: 1, 2].
* **Backend:** Java Spring Boot[cite: 1, 2].
* **Metodologías:** Clean Code, POO[cite: 1, 2].
